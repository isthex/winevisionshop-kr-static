국민은행 공동인증서란?

KB공동인증서와 KB국민인증서 차이점

KB공동인증서는 금융결제원 인증 체계를 따르는 범용 인증서로, 홈택스, 정부24, 건강보험공단 등에서 사용 가능합니다. 반면 KB국민인증서는 국민은행 전용 간편인증서로, 공동인증서 대비 인증 절차가 단순하며 국민은행 전용 서비스에서 주로 사용됩니다.

발급이 필요한 주요 상황

  • 정부기관 사이트에서 전자서명 시
  • 타 은행/보험/증권 거래를 위한 범용 인증서 필요 시
  • 연말정산, 종합소득세 신고 등 공공서비스 이용 시

국민은행 공동인증서 발급 방법

KB인증서발급 모바일 앱 절차

  1. 국민은행 앱(KB스타뱅킹) 실행 → 로그인
  2. 인증센터 메뉴 → 공동인증서 발급/재발급 선택
  3. 본인확인(OTP 또는 휴대폰 인증) 진행
  4. 공동인증서 비밀번호 설정 → 발급 완료

발급한 공동인증서는 스마트폰 내 저장소 또는 클라우드(공동인증서 저장소) 선택이 가능합니다.



PC에서 공동인증서 발급하는 방법

  1. KB국민은행 인터넷뱅킹 접속 → 로그인
  2. 개인 > 인증센터 메뉴 클릭
  3. ‘공동인증서 발급/재발급’ 선택
  4. 보안매체 확인 후 공동인증서 발급 완료

PC 환경에서는 브라우저 보안 프로그램이 필요하며, 크롬·엣지·인터넷 익스플로러 모두 지원됩니다.

KB국민인증서 갱신 및 재발급

인증서 만료 전 갱신 절차

공동인증서는 1년의 유효기간을 가지며, 만료 전에는 KB스타뱅킹 앱 또는 인터넷뱅킹에서 간편하게 KB국민인증서 갱신을 할 수 있습니다.

  1. KB스타뱅킹 > 인증센터 > 인증서 갱신
  2. 기존 인증서 확인 → 인증서 비밀번호 입력
  3. 새로 갱신된 인증서 다운로드 완료

유효기간 경과 후 재발급 방법

유효기간이 지난 경우에는 기존 인증서를 사용할 수 없으며, 새로 발급 절차를 다시 진행해야 합니다. 앱이나 PC에서 동일한 절차로 재발급 가능합니다.

KB국민인증서 내보내기 및 복사

스마트폰 → PC 인증서 내보내기

  1. KB스타뱅킹 실행 > 인증센터 > 인증서 내보내기
  2. PC에서 KB국민은행 인터넷뱅킹 로그인
  3. PC 화면에 표시된 인증코드 → 모바일에서 입력
  4. 내보내기 완료 후 PC에서 인증서 사용 가능

다른 기기 간 인증서 복사 방법

  • 모바일 간 복사: 앱 내 인증서 복사 기능 사용
  • PC 간 복사: 인증서 폴더(.NPKI)를 USB에 저장하여 이동

내보내기 기능은 보안성을 강화한 클라우드 인증서에도 적용됩니다.

자주 묻는 질문(FAQ)

Q1. KB인증서는 공동인증서와 무엇이 다른가요?

A. KB공동인증서는 범용 인증서로 정부 및 타 금융기관에서도 사용 가능하며, KB국민인증서는 국민은행 전용 간편인증서입니다.

Q2. KB국민인증서 내보내기는 어떻게 하나요?

A. KB스타뱅킹 앱의 ‘인증서 내보내기’ 메뉴에서 PC 인증코드 입력 방식으로 간편하게 전송 가능합니다.
